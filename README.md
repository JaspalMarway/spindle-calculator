# Spindle Calculator

## Publish to GitHub Pages

1. Create a new public GitHub repository called `spindle-calculator`.
2. Upload all files in this folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select **main** branch and **/(root)**.
6. Save.
7. After a minute your site will be available at:

https://YOUR-GITHUB-USERNAME.github.io/spindle-calculator/

On iPhone, open the site in Safari and choose **Share > Add to Home Screen**.
